Help How to use Edge Diagrammer

Step One


Run RunEdgeConvert.java




Step Two


Click “File” and Hover “Open Edge File”




Step Three


Open Courses.edg




Step Four



If click “Student”, either click “StudentSSN” or “StudentName”




Step Five


If choose “StudentSSN”, click “Integer” and “Set Default Value” to input the number, which is 9 since there are only 9 digits for social security.




Step Six


After defining the tables, define the relations among tables by clicking the ‘Define Relations’ button.




Step Seven


From the Define Relations UI, you are able to create new relations among fields from various tables or erase existing relations.

Step Eight


Next, click on the ‘Options’ menu and select ‘Set Output File Definition Location’ and select a folder you would like the output files to be saved.

Step Nine


Finally, click the ‘Create DDL’ button to convert the database statements and save as appropriate database script file.

Notes


You may save your progress at any time by clicking the ‘File’ menu tab and selecting ‘Save’ or ‘Save as’.

You may open a saved file that is still currently in progress by clicking the ‘File’ menu tab and selecting ‘Open Save File’.
